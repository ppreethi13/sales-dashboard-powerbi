# Sales Analytics Dashboard — Power BI

##  Project Overview
This project presents an interactive Sales Analytics Dashboard built using Power BI to transform raw sales data into meaningful business insights. The dashboard enables users to monitor performance, identify trends, and support data-driven decision making.

##  Objective
- Analyze sales performance across regions and product categories  
- Track key business metrics such as revenue and profit  
- Identify trends and patterns in sales data  
- Enable quick and intuitive data exploration  

##  Key Features
-  KPI Cards displaying Total Sales and Profit  
-  Category-wise and region-wise performance analysis  
-  Time-series visualization to track sales trends  
-  Interactive slicers for dynamic filtering  
-  Drill-down capability for deeper insights  

##  Tech Stack
- Power BI (Dashboard Development & Visualization)  
- CSV Dataset (Data Source)  

##  Dataset Description
The dataset includes the following fields:
- Order Date  
- Region  
- Product Category  
- Sales Amount  
- Profit  

##  Insights Generated
- Identified top-performing regions contributing maximum revenue  
- Observed trends in sales over time  
- Highlighted high-performing product categories  
- Detected variations in profit across categories  

##  Dashboard Design Approach
- Applied clean UI/UX principles for better readability  
- Used consistent color themes and layout hierarchy  
- Focused on clarity, simplicity, and user experience  

##  How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Use slicers and filters to explore insights
